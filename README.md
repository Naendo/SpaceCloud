## About

> **SpaceCloud** handles the **actions** connected to **hosting** a coworking.
> The owner of a coworking is able to **create** his **individual products** like meeting rooms, fixdesks and so on.
> Users can create accouts which are connected to this coworking and are able to easily book products or
> get the information they need. **SpaceCloud** also has a implemented invoicing
> system which makes the whole booking procedure automated.


## Deployment

> We use [Docker](https://hub.docker.com/) combined with [Docker-Compose](https://docs.docker.com/compose/) to make our project build ready.
> [DigitalOcean](https://www.digitalocean.com/) provides us with a *docker-ready* server.
